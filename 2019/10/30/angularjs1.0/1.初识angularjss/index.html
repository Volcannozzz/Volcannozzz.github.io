初识angularjs

该框架是一个jsmvc框架，具备以下特点：

  1. 把应用程序数据绑定到 HTML 元素；
  2. 可以克隆和重复 HTML 元素。
  3. 可以隐藏和显示 HTML 元素。
  4. 可以在 HTML 元素”背后”添加代码。
  5. 支持输入验证。
